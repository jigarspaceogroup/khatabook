## Querying Sample Data

### Get all customers with balances

```sql
SELECT
  k.name as khatabook,
  c.name as customer,
  c.phone_number,
  c.opening_balance / 100.0 as opening_rs,
  c.current_balance / 100.0 as balance_rs,
  c.last_transaction_at
FROM customers c
JOIN khatabooks k ON c.khatabook_id = k.id
WHERE c.deleted_at IS NULL
ORDER BY c.current_balance DESC;
```

### Get transaction history for a customer

```sql
SELECT
  t.transaction_date,
  t.type,
  t.amount / 100.0 as amount_rs,
  t.payment_mode,
  t.note
FROM transactions t
WHERE t.customer_id = '770e8400-e29b-41d4-a716-446655440002'
  AND t.deleted_at IS NULL
ORDER BY t.transaction_date DESC;
```

### Get invoice with items

```sql
SELECT
  i.invoice_number,
  i.invoice_date,
  c.name as customer,
  ii.item_name,
  ii.quantity,
  ii.rate / 100.0 as rate_rs,
  ii.tax_rate,
  ii.total / 100.0 as total_rs
FROM invoices i
JOIN customers c ON i.customer_id = c.id
JOIN invoice_items ii ON ii.invoice_id = i.id
WHERE i.deleted_at IS NULL
ORDER BY i.invoice_date DESC, ii.sort_order;
```
